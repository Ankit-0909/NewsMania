# Newsmania

A simple news website built with pure HTML, CSS, and JavaScript. News is added manually through an admin panel (no external APIs used).

Tech Stack
Java Spring Boot(Backend)
HTML5
CSS3
Vanilla JavaScript
localStorage for data persistence

## Features
- Admin panel to add news (title, description, image, category)
- Display news on the home page
- Category-based filtering
- Fully static and lightweight
- Data stored in browser's localStorage
- Responsive design
